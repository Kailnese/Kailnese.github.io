37. Valid Sudoku解析

Posted on 2020-01-12 Edited on 2020-01-13 In leetcode Views:

这道题其实不难. 只要知道sudoku的方法就好了. 检查一排间有没有重复数字, 检查一列有没有重复数字. 检查9个大格子中有没有重复数字.

这个大格子用一个hashmap储存<每个大格子的index, Integer>.

index的计算则用index = (row/3) * 3 + col/3

row除以3, 则能把9个row平均分成3份 0, 1, 2. 但是我们需要的是0, 3, 6, 则乘以3. 后面col除以3, 则可以把列分成3份. 加起来就是index了

剩下的就是2个for loop的问题了. 因为sudoku的格子大小固定, n是常数, 所以时间复杂度O(1)
