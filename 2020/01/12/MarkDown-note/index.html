Basic MarkDown command

  • - && 1. 是list符号
  • 如果要创建new line, 则按两次回车
  • 如果要加link在一个词或句子中则放’[]()‘中括号, 后面小括号放link
  • 如果要higlight某个词, 则用键盘左上角的点来围起来
  • 如果要创建code block, 则用三个键盘左上角的点, 如下所示
  • 如果要插入图片, 则用跟link相同的符号![alt text](picture link) For example:
  • Some paragraph with text, 使用>来输出blockquote

    hello there

  • 创建table的用|符号
    • |Column 1|| Column 2|| Column 3|
    • | --- | --- | --- | (这一行必须要, 拿来分断column title和content用的)
      Column 1 Column 2 Column 3
      Content 1 Content 2 Content 3
  • 如果要让某个单词bold, 则用**bold**, 或者让某个单词italic, 则只用一个星号框起来就好了. 再或者如果要体现remove的效果, 用两个波浪线~~圈起来
